# 🐾 Pet Content Writer — 宠物文章创作 Skill

> 从选题到成稿的全流程宠物内容创作工具。
> 输出可直接用于小红书发布的完整文案。

---

## 触发条件

**关键词触发**：
- `写宠物文章` / `宠物选题` / `宠物内容` / `pet content`
- `今天写什么` / `今天的宠物文章`
- `帮我写一篇关于XX的宠物文章`

**场景触发**：
- 用户说"今天发什么宠物内容"
- 用户给出一个宠物话题，需要创作文章
- 每日运营 SOP 中的 Step 4（内容创作）

---

## 前置依赖

- **角色规范**：遵循 `xhs-pet-operator.mdc` 中定义的写作风格和内容规范
- **运营日志**：读取 `.codebuddy/memory/pet-operation-log.md` 进行内容去重
- **小红书 MCP**：可选，用于热点参考

---

## 执行流程

### Phase 1：选题确定

#### 1.1 检查运营日志
读取 `.codebuddy/memory/pet-operation-log.md`，了解：
- 近 7 天发布了什么主题
- 近期内容类型分布（避免连续同类型）
- 哪些类型还没覆盖到

#### 1.2 选题来源（按优先级）

**来源 A：用户指定选题**
用户直接给出话题 → 跳到 Phase 2

**来源 B：热点驱动选题**
调用小红书 MCP 搜索当前热门宠物内容：
```
search_feeds(keyword="猫咪", filters={sort_by:"最新"})
search_feeds(keyword="养猫日常", filters={sort_by:"最多点赞"})
search_feeds(keyword="宠物性格", filters={sort_by:"最新"})
```
从热门内容中提取可借鉴的选题角度。

**来源 C：内容日历选题**
根据内容方向优先级轮转：

| 周一 | 周二 | 周三 | 周四 | 周五 | 周六 | 周日 |
|------|------|------|------|------|------|------|
| 性格分析 | 行为解读 | 养宠趣事 | 知识科普 | 性格分析 | 产品引流 | 自由选题 |

#### 1.3 选题输出
确定选题后，输出：
```
📋 今日选题
- 主题：{主题描述}
- 类型：{性格分析/行为解读/养宠趣事/知识科普/产品引流}
- 角度：{具体切入角度}
- 目标读者：{谁会对这个感兴趣}
```

---

### Phase 2：标题创作

#### 2.1 生成 3 个候选标题
基于选题，用不同公式生成 3 个标题：

```
标题候选：
1. [反差萌] {标题1}
2. [提问式] {标题2}
3. [共鸣式] {标题3}

推荐：第 X 个（理由：xxx）
```

#### 2.2 标题检查
- [ ] 长度 ≤ 20 中文字
- [ ] 不含违禁词
- [ ] 有吸引力，能在信息流中脱颖而出
- [ ] 与近 30 天标题不重复

---

### Phase 3：正文创作

#### 3.1 正文结构
遵循 `xhs-pet-operator.mdc` 中的正文模板：

```
[场景引入 / 趣味开头] 🐱

[核心内容 - 知识/观点/故事]
- 要点1
- 要点2
- 要点3

[延伸/深入/反转]

[互动引导] 💬
你家毛孩子是什么性格？评论区聊聊～
```

#### 3.2 正文规范
- 300-600 字
- 每段不超过 3 行
- 适度使用 emoji（每段 1-2 个）
- 语气像朋友聊天
- 不含 # 话题标签（标签单独处理）

#### 3.3 内容类型特化模板

**性格分析类**：
```
你知道吗？{宠物行为} 其实说明你家毛孩子是{性格类型}！🐾

{性格特点描述}
- 特点1：{描述}
- 特点2：{描述}
- 特点3：{描述}

{有趣的比喻/拟人化描述}

想知道你家毛孩子的完整性格报告吗？
试试宠物MBTI测试 → microlab.top 💕

你家宝贝是什么性格？评论区告诉我～
```

**行为解读类**：
```
{常见宠物行为场景} 你遇到过吗？😂

其实这个行为背后的含义是...
{科学解释，但用通俗语言}

1️⃣ {解读1}
2️⃣ {解读2}
3️⃣ {解读3}

下次再看到你家{猫/狗}这样做，你就知道它在想什么了 🧠

你家毛孩子有什么让你困惑的行为？评论区聊聊～
```

**养宠趣事类**：
```
{一个有趣的场景/故事开头} 😹

{展开故事/描述}

{转折或高潮}

{感悟/总结}

养{猫/狗}就是这样，每天都有新惊喜 ❤️
你们有类似的经历吗？
```

---

### Phase 4：标签推荐

#### 4.1 标签生成规则
从标签池中选择 3-5 个标签：
- 1 个核心标签（宠物MBTI / 猫咪性格测试 / 宠物性格）
- 1-2 个流量标签（养猫日常 / 猫咪 / 萌宠）
- 1-2 个精准标签（与文章内容强相关）

#### 4.2 标签输出
```
推荐标签：["标签1", "标签2", "标签3", "标签4"]
```

---

### Phase 5：配图建议

#### 5.1 配图方案
根据内容类型推荐配图：

| 内容类型 | 推荐配图 |
|----------|----------|
| 性格分析 | 对应性格的猫/狗照片，表情要匹配性格特点 |
| 行为解读 | 展示该行为的宠物照片 |
| 养宠趣事 | 搞笑/温馨的宠物照片 |
| 知识科普 | 清晰的示意图或可爱的宠物照片 |
| 产品引流 | 测试结果截图 + 宠物照片 |

#### 5.2 图片来源建议
```
配图建议：
- 方案A：使用网络图片 URL（推荐 Unsplash/Pexels 搜索 "{关键词}"）
- 方案B：用户提供本地图片
- 方案C：AI 生成宠物插画

推荐搜索词：{英文搜索关键词}
示例 URL：https://unsplash.com/s/photos/{keyword}
```

---

### Phase 6：完整输出

将所有内容整合为发布就绪的格式：

```
═══════════════════════════════════════
📝 小红书宠物文章 — 发布就绪
═══════════════════════════════════════

📌 标题：{最终标题}

📄 正文：
{完整正文内容}

🏷️ 标签：["标签1", "标签2", "标签3"]

🖼️ 配图：
- {配图方案描述}
- {图片路径或 URL}

═══════════════════════════════════════
⚡ 确认后将调用发布 Skill 发布到小红书
═══════════════════════════════════════
```

## 输入输出规范

### 输入
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 选题 | 文本 | ❌ | 指定选题方向，不提供则自动选题 |
| 类型 | 枚举 | ❌ | 性格分析/行为解读/养宠趣事/知识科普/产品引流 |
| 图片 | 路径/URL | ❌ | 用户提供的配图，不提供则给出建议 |

### 输出
| 产物 | 说明 |
|------|------|
| 标题 | ≤20字的小红书标题 |
| 正文 | 300-600字的完整正文 |
| 标签 | 3-5个话题标签数组 |
| 配图建议 | 配图方案和来源建议 |

---

## 质量标准

一篇好的宠物文章应该满足：

1. **有趣**：读者看完会心一笑或产生共鸣
2. **有用**：提供了某个知识点或新视角
3. **有互动性**：结尾有引导评论的问题
4. **有辨识度**：符合账号人设，不是通用模板文
5. **有传播性**：读者愿意分享给其他养宠朋友
