# 🧠 AI 知识库

> 持续学习、总结、沉淀 —— 你的 AI 学习助手产出的所有知识，都有一个统一的"家"

## 目录结构

```
knowledge/
├── articles/      # 文章学习笔记（公众号、博客等）
├── insights/      # 行业洞察与深度分析
└── playbooks/     # 方法论、经验手册
```

## 子目录说明

### 📖 articles/ — 文章学习笔记

通过 `@wechat-article-learner` skill 总结的公众号文章、博客等学习笔记。

**文件命名规范**：`YYYY-MM-DD-文章主题.md`

**示例**：
- `2026-03-26-agent-first-范式革命.md`
- `2026-03-28-RAG-最佳实践.md`

### 🔍 insights/ — 行业洞察与深度分析

深度分析报告、竞品研究、技术趋势、产品/设计/开发阶段的角色分析等。

**当前内容**：
- GStack 深度分析文档
- IMA 研发流程 AI 辅助方案
- 产品构思/设计/开发阶段的角色分析

### 📘 playbooks/ — 方法论与经验手册

实操经验、工作流程、思维模型、变现思路等可复用的方法论。

**当前内容**：
- AI 写作思路经验
- AI-Agent 时代个人红利与变现思路
- 总思路与经验文档列表

## 使用方式

### 添加文章笔记

```
@wechat-article-learner 总结下这篇文章：<URL>
```

Skill 产出的学习笔记会保存到 `articles/` 目录。

### 手动添加知识

直接在对应子目录创建 `.md` 文件即可。建议遵循命名规范，方便后续检索。

## 知识检索

所有 `.md` 文件都可以被 AI 助手直接读取和引用，形成你的个人知识图谱。
