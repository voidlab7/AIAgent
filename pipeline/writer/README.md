# 模块4：写稿 (Content Writer)

## 功能描述
AI 辅助内容创作，从提纲到成稿的完整流程。

## 输入
- 选题信息（标题、角度）
- 素材库（来自模块3）
- 内容偏好（字数、风格）

## 输出
- **Markdown 初稿**
- **多版本**（可选）：不同风格、不同长度
- **SEO 关键词**

## 核心能力
1. **提纲生成**：AI 生成文章结构
2. **段落扩写**：基于素材逐段创作
3. **全文润色**：语言优化、逻辑检查
4. **风格迁移**：适配不同平台风格
5. **原创检测**：AI 自检 + 人工复查

## 文件结构

## 使用方式

### 生成提纲
```bash
python -m 4_内容创作.main --topic-id 1 --step outline
```

### 完整创作流程
```bash
python -m 4_内容创作.main --topic-id 1 --full
```

### 多版本生成
```bash
python -m 4_内容创作.main --topic-id 1 --versions 3
```

## Skill 规划
**Skill 名称**：`content-creator.md`

**核心工作流**：
1. **读取上下文**：选题 + 素材库
2. **生成提纲**：
   - AI 分析素材
   - 生成逻辑清晰的大纲（3-5 个小节）
3. **逐段扩写**：
   - 每个小节调用 AI 生成 300-500 字
   - 插入素材（数据、金句、案例）
4. **全文润色**：
   - 语言优化（去口水话、增强可读性）
   - 逻辑检查（衔接、重复）
5. **风格适配**（可选）：
   - 微信：正式、深度
   - 小红书：轻松、表情包
6. **SEO 优化**：提取关键词
7. 存入 `articles` 表
