# 🏭 内容创作流水线

> 从热点发现到内容发布的全自动化 AI 内容创作管线

## 流水线架构

```
pipeline/
├── hotspot/       # 1. 热点监控 — 实时抓取各平台热搜
├── topic/         # 2. 选题筛选 — 基于热点生成选题
├── material/      # 3. 素材采集 — 多源内容聚合
├── writer/        # 4. 内容创作 — AI 辅助写稿
├── image/         # 5. 配图查找 — 自动匹配配图
├── formatter/     # 6. 格式优化 — 多平台格式适配
└── publisher/     # 7. 内容发布 — 自动化发布
```

## 模块说明

### 1. hotspot/ — 热点监控 🚧

**功能**：实时抓取并分析各平台热搜榜单

- **数据源**：微博热搜、知乎热榜、小红书热门（RedNote MCP）、抖音热榜、百度风云榜
- **输出**：热点关键词、热度值、趋势分析
- **热度公式**：`likes * 1 + favorites * 1.5 + comments * 2 + shares * 3`

### 2. topic/ — 选题筛选 🚧

**功能**：基于热点和账号定位生成选题

- **输入**：热搜数据 + 账号画像 + 历史数据
- **输出**：选题列表（标题、角度、预期流量）
- **已实现**：趣味测试分析器 + 关键词触发

### 3. material/ — 素材采集 📋

**功能**：根据选题搜集并整理素材

- **数据源**：搜索引擎、知识库、竞品内容、评论区
- **输出**：结构化素材库（观点、数据、案例、金句）

### 4. writer/ — 内容创作 📋

**功能**：AI 辅助内容创作

- **模式**：提纲生成 → 段落扩写 → 全文润色
- **输出**：Markdown 初稿 + 多版本
- **内容长度**：微信 1500-3000 字 / 小红书 300-800 字 / 知乎 1000-2500 字

### 5. image/ — 配图查找 📋

**功能**：自动匹配文章配图

- **来源**：Unsplash、Pexels、AI 生成
- **输出**：配图 URL + 本地缓存

### 6. formatter/ — 格式优化 📋

**功能**：多平台格式转换和美化

- **支持**：微信公众号、小红书、知乎、头条号
- **输出**：平台专属 HTML/Markdown

### 7. publisher/ — 内容发布 🚧

**功能**：自动化内容发布

- **平台**：微信公众号（wechat-publisher-mcp）、小红书（RedNote MCP）
- **模式**：定时发布、预览审核
- **小红书限制**：标题 ≤ 20 中文字，话题标签用 tags 数组
